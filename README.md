# Augmented Notes

Augmented Notes is an English-language site/blog about immersive technologies:
AR, VR, MR, XR, haptics, devices, use cases, and curated industry news.

## Local Development

Install dependencies:

```sh
npm install
```

Start the site:

```sh
npm run dev
```

Build for production:

```sh
npm run build
```

## Content

Posts live in `src/content/posts`.

News sources and curated links live in `src/data`.
